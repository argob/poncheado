Tablas

Tabla simple

Para mostrar un dato por columna. Se usa para cuando hay pocas columnas.

Tabla con columnas combinadas

Para mostrar varios datos por columna. Se usa para cuando hay muchas columnas.

Tabla con filtro de texto, paginado y ordenamiento (Data Tables)

Cuando se necesitan mostrar muchas filas es conveniente usar un filtro que permita buscar texto sobre cualquier elemento de la tabla. Para eso se usa el componente "Data Tables" que además permite ordenar por columnas y le agrega un paginado automáticamente.

Tabla responsive (cambia en vista mobile)

Para mostrar uno o varios datos por columna. Se usa para cuando hay muchas columnas y se necesita priorizar la vista en dispositivos móviles.

Tabla con scroll horizontal

Sólo ser usado para casos con muchas columnas que no puedan combinarse (grilla de horarios o tarifas). Este comportamiento debe evitarse ya que no es cómodo para dispositivos móviles.
