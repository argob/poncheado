Paneles

Panel simple (sólo título)

Para cuando se tienen que listar muchos paneles (más de 20) o para cuando el título es suficientemente descriptivo.

Panel simple con texto (título y bajada)

Para cuando se tienen que listar una cantidad moderada de paneles (menos de 20).

Panel simple con ícono y texto

Para listar pocos paneles (8 o 12) y se quiere diferenciar mejor un panel del otro.

Panel destacado con ícono y fondo de color

Para listar pocos paneles (2 a 8) y se quiere darle mucha jerarquía a los paneles.

Panel destacado con imagen

Igual al caso anterior pero en vez de usar un ícono con fondo de color, se usa una imagen de fondo.

Panel desactivado
