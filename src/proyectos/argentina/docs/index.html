Atajos

Los atajos se utilizan para enlazar cualquier tipo de contenido, tanto interno como externo, y con diferentes tipos de jerarquías.

Atajo simple

El atajo siemple consta de los siguientes campos:

  • Título
  • Texto descriptivo (opcional)
  • Link interno o externo
  • Ícono (opcional)
  • Color de ícono (opcional, de manera predeterminada se muestra azul)

Listados

Los listados obtienen la información de un tipo de contenido y un group audience específico de Drupal. Tienen la capacidad de seleccionar la cantidad de elementos a mostrar, paginarlos y agregar un botón para ver más.

Listados en panelizer

Los listados tienen la siguiente configuración:

  • Título
  • Cantidad de elementos a mostrar
  • Tipo de listado:
    • Últimos cargados
    • Relacionados
    • Personalizado
  • Tipos de contenidos que se pueden elegir: (solo aplica para los tipos de listados últimos cargados y relacionados)
    • Páginas
    • Noticias (con capacidad de elegir una categoría)
    • Servicios (con capacidad de elegir una categoría)
    • Áreas
    • Aplicaciones
  • Contenido personalizado (solo aplica para el tipo de listado personalizado)
  • Formato de visualización
    • Solo Título
    • Título y texto
    • Título e imagen
    • Título, texto e imagen
    • Noticias
    • Noticias destacadas
    • Servicios ???????????????????????
  • Mostrar el botón "Ver todos"
  • Mostrar paginador
  • Orientación ???????????????????????

Alertas

Las alertas se pueden utilizar de dos formas en drupal: como un módulo de panelizer, o como un text filter en cualquier campo que tenga markdown.

Módulo de alerta en panelizer

El módulo se agrega en panelizer dentro de argentina / alerta, y tiene la siguiente configuración:

  • Título (opcional, acepta markdown)
  • Texto descriptivo (acepta markdown)
  • Ícono (opcional)
  • Color de la alerta (opcional, de manera predeterminada se muestra amarillo)
¿Ya iniciaste tu tramite?

Seguí el estado del trámite en linea

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Dolorem ratione, illo quaerat corporis, suscipit tenetur vel iusto ullam ex ipsa ipsam recusandae nulla, accusamus officia sed! Dolores itaque expedita, repudiandae.

Alertas en Markdown

En markdown se puede agregar una alerta y tiene las siguiente configuración:

  • Título (opcional, acepta markdown)
  • Texto descriptivo (acepta markdown)
  • Ícono (opcional)
  • Color de la alerta (opcional, de manera predeterminada se muestra amarillo)
¿Ya iniciaste tu tramite?

Seguí el estado del trámite en linea.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Dolorem ratione, illo quaerat corporis, suscipit tenetur vel iusto ullam ex ipsa ipsam recusandae nulla, accusamus officia sed! Dolores itaque expedita, repudiandae.

 
{ alerta | color: amarillo | icono: check | titulo: ¿Ya iniciaste tu tramite? | texto: Seguí el [estado del trámite en linea](http://www.mininterior.gov.ar/tramitesyservicios/estado-tramite-dni.php). }

{ alerta | color: verde | texto: Lorem ipsum *dolor sit amet*, consectetur adipisicing elit. Dolorem ratione, illo quaerat corporis, suscipit tenetur vel iusto ullam ex ipsa ipsam recusandae nulla, [accusamus officia sed](#)! Dolores itaque expedita, repudiandae. }

Ejes

Los ejes son contenidos breves que se muestran en columnas.

Ejes en markdown

El contenido de los ejes se compone de:

  • Título (acepta markdown)
  • Text (acepta markdown)
  • Ícono (opcional)
  • Color del ícono (opcional, de manera predeterminada se muestra celeste)

Cercana

Conoce tus necesidades y adopta sus servicios a ellas.

Transparente

Brinda mayor información y certeza sobre sus actos de gobierno.

Participativa

Trabaja en equipo con los empleados públicos y la gente.

 
{ ejes: 
  { titulo: Cercana | texto: Conoce tus necesidades y adopta sus servicios a ellas. | icono: fa-users | color: verde },
  { titulo: Transparente | texto: Brinda mayor información y certeza sobre sus actos de gobierno. | icono: fa-search | color: azul },
  { titulo: Participativa | texto: Trabaja en equipo con los empleados públicos y la gente. | icono: fa-users } 
}

Números destacados

Los números destacados se utilizan en landings, páginas o noticias que necesitan comunicar un resultado o medición.

Números destacados en markdown

El contenido se compone de:

  • Número (solo texto, máximo 10 caracteres)
  • Medida (solo texto, máximo 20 caracteres)
  • Descripción (solo texto)
  • Tamaño (del 1 al 3, de manera predeterminada se usa 2)
  • Color (opcional, de manera predeterminada se muestra negro)
392.493

Visitas últimos 30 días

+11,06% vs mes anterior

33 %

Visitas este mes

del 1 al 26 de Febrero

5 de 10

Personas conectadas

Duramente los últimos 30 días.

 
{ numeros: 
  { numero: 392.493 | titulo: Visitas últimos 30 días | texto: +11,06% vs mes anterior | color: verde },
  { numero: 33 | medida: % | titulo: Visitas este mes | texto: del 1 al 26 de Febrero | color: gris },
  { numero: 5 | medida: de 10 | titulo: Personas conectadas | texto: Duramente los últimos 30 días. } 
}

Citas

Las citas pueden ser estándares o destacadas.

Citas en markdown

El contenido de las citas se compone de:

  • Text (acepta markdown)
  • Autor (opcional, acepta markdown)

“ Un Estado inteligente con servidores públicos capacitados para brindarle más y mejores servicios a los ciudadanos. Esto es el Siglo XXI y en eso trabajaremos día y noche. ”

Mauricio Macri

Referencias

Colores de texto

Los atajos se utilizan para enlazar cualquier tipo de contenido, tanto interno como externo, y con diferentes tipos de jerarquías.

Paleta principal

la paleta principal se forma de:

  • Azul (.text-primary)
  • Celeste (.text-secondary)
  • Verde (.text-success)
  • Amarillo (.text-warning)
  • Rojo (.text-danger)
  • Gris claro (.text-muted)
  • Gris oscuro (.text-gray)
  • Negro (por defecto)

Paleta extendida

El atajo siemple consta de los siguientes campos:

  • Fucsia (.text-fucsia)
  • Arandano (.text-arandano)
  • Uva (.text-uva)
  • Cielo (.text-cielo)
  • Verdin (.text-verdin)
  • Lima (.text-lima)
  • Maiz (.text-maiz)
  • Tomate (.text-tomate)

Colores de fondo

Los atajos se utilizan para enlazar cualquier tipo de contenido, tanto interno como externo, y con diferentes tipos de jerarquías.

Paleta principal

El atajo siemple consta de los siguientes campos:

.bg-primary .bg-success .bg-info .bg-warning .bg-danger .bg-gray .bg-white

Paleta extendida

El atajo siemple consta de los siguientes campos:

.bg-fucsia .bg-arandano .bg-uva .bg-cielo .bg-verdin .bg-lima .bg-maiz .bg-tomate
