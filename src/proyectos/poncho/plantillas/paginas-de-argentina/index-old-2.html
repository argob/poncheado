Página

Página genérica para mostrar diferentes contenidos para los cuales no existe un tipo de contenido específico en Drupal. Se usa para:

  • Landings promocionales de una campaña o un servicio complejo
  • Agrupar diferentes servicios
  • Contenidos con formatos nuevos o especiales

Página de libro

Página que agrupa otras páginas mediante una navegación lateral (sidebar). Esto permite ver contenidos de un mismo tema e ir y volver entre páginas con comodidad. Los contenidos de estas páginas deben ser simples y no tener la misma complejidad que los tipos de contenido Página.
