Redacción

Redacción

Poncho y los sitios del Gobierno respetan ciertos principios para desarrollar textos de contenidos o de la interfaz.

Principios de redacción

Tanto para cuerpos de texto (de instructivos, noticias, etc) como para elementos de interfaz (botones, formularios, tablas, etc) se tienen en cuenta los siguientes principios:

  • Textos cortos (sin usar palabras de más)
  • Textos simples (usando palabras fáciles, sin lenguaje técnico)

Jerarquías de texto

Cuando una página tiene un cuerpo de texto, el mismo debe estar separado por párrafos y encabezados para una mejor lectura.

Párrafos

Los párrafos están compuestos por una o varias oraciones que hablan de una misma idea. Cuando se quiere seguir hablando del mismo tema pero detallar otra idea, se usa otro párrafo.

Encabezados

Los encabezados sirven para agrupar párrafos que hablan de diferentes temas. Al usar encabezados el usuario también puede escanear rápidamente los temas de una página antes de empezar a leer los párrafos.

Siempre debe respetarse el orden (h1 al h5) y no saltearse encabezados.
