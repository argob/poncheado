Detalle de elementos

Encabezado

Usar este encabezado para agrupar los datos y separarlos de otros para una lectura más cómoda.

País de residencia

Sirve para saber en qué país reside el ciudadano, para cuando el servicio también lo pueden usar argentinos o extranjeros residentes en el exterior.

Funcionamiento

Si se cambia este campo, los de abajo (provincia, partido o departamento y localidad) se reseteean y deshabilitan. La primera opción del listado es Argentina.

Provincia

Sirve para saber en dónde reside el ciudadano, para cuando el servicio se ofrece por diferentes canales según el lugar.

Funcionamiento

Si se cambia este campo, los de abajo (partido o departamento y localidad) se reseteean y deshabilitan.

Los datos son tomados de la API del servicio de normalización y geocodificación para direcciones de Argentina.

Partido o departamento

Sirve para saber en dónde reside el ciudadano, para cuando el servicio se ofrece por diferentes canales según el lugar.

Funcionamiento

Si se cambia este campo, el de abajo (localidad) se reseteea y deshabilita.

Los datos son tomados de la API del servicio de normalización y geocodificación para direcciones de Argentina.

Localidad

Sirve para saber en dónde reside el ciudadano, para cuando el servicio se ofrece por diferentes canales según el lugar.

Funcionamiento

Ingresá una localidad

Los datos son tomados de la API del servicio de normalización y geocodificación para direcciones de Argentina.

Código postal

Sirve para identificar y llamar al usuario por su nombre en páginas donde esté loggeado o correos electrónicos que se le envíen.

Funcionamiento

Si el usuario completa el código postal, se autocompletan los campos Provincia, Partido o departamento y Localidad. Para eso debe usarse el servicio de provincias, partidos o departamentos y localidades.

Calle

Sirve para comunicarse con el ciudadano por correo tradicional o para que el agente de un organismo se acerque al domicilio del ciudadano.

Funcionamiento

Está en la misma fila a la izquierda de Número de calle.

Número de calle

Complementa al campo Calle.

Funcionamiento

Está en la misma fila a la derecha de Calle.

Piso

Complementa al campo Calle y número.

Departamento

Complementa al campo Calle y número.

Otros datos del domicilio

Complementa a los campos Calle y número, y Piso y departamento. Sirve para indicar parcela u otra referencia de ubicación del domiclio. Este campo nunca es obligatorio.
