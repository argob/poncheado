Las alertas se dividen por color de acuerdo a su función y combinan las siguientes variables: simples (sólo texto), con título, íconos, botones, enlaces y pueden ser colapsable.

Uso según el color en alertas simples

Azul para alertas de información
Verde para alertas de éxito
Amarillo para alertas de advertencia.
Rojo para alertas de peligro o error en proceso
